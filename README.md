### Usage
Move your kube config to `~/.kube/config` and run `tdsd` in your terminal.

```bash
$ tdsd $project_name
```

**Example:**
If u have a config file named `salo.yaml` in `.kube` folder and u want to switch to it, just run:

```bash
$ tdsd salo
```
